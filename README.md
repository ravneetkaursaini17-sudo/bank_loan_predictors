# Bank Loan Approval Simulator

A Python-based financial application that simulates how banks evaluate loan applications using borrower credit metrics and underwriting logic.

The tool analyzes borrower profiles using:

- Debt-to-Income (DTI) ratios
- FICO credit score tiers
- Loan affordability metrics
- Risk-based interest rate pricing

Users can modify borrower inputs and see how approval probability and loan terms change.

---

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy

## How to Run the App

Clone the repository

```
git clone https://github.com/ravneetkaursaini17-sudo/bank_loan_predictors.git
```

Install dependencies

```
pip install -r requirements.txt
```

Run the Streamlit app

```
streamlit run app.py
```
